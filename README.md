# odin-signup-form
Recreating the frontend of a sign-up form. An assignment from The Odin Project (Full-Stack Javascript Path).

## Overview
This small project is basically just recreating the ui of a signup form, and enforcing constraints to the form inputs so that the required fields are filled and displaying visual cues of that validation. Both password fields check for a value that has at least 8 characters, a lower case and uppercase and a number. However, it does not check if both are the same as that is out of the scope of this project and will be approached in a future one.
